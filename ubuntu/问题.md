#### 远程登陆 远程失败  远程登录失败
```shell

vim /etc/ssh/sshd_config
# 将/etc/ssh/sshd_config文件中的PermitRootLogin prohibit-password注释掉，

PermitRootLogin yes
# 设置完成后重新启动SSH服务

sudo service ssh restart
```
